---

## STEP 4: TECHNICAL SEO

- **Sitemap:** Created at `/sitemap.xml`
- **Robots.txt:** Created at `/robots.txt`
- **Canonical Tags:** Implemented via `SEO` component.
- **Schema Markup:** JSON-LD for `WebSite` and `Organization` implemented.
- **Open Graph:** Meta tags for Facebook/Twitter implemented.

---

## STEP 5: PERFORMANCE OPTIMIZATION

- **Lazy Loading:** All images use `loading="lazy"`.
- **WebP:** Recommended for all future image uploads.
- **Core Web Vitals:** Minimized JS execution by using functional components and efficient hooks.
- **CDN:** Netlify automatically provides global CDN and caching.

---

## STEP 6: INTERNAL LINKING

- **Navbar:** Links to all major category pages.
- **Footer:** Links to all categories and support pages.
- **Homepage:** Links to top 10 sections and full category pages.
- **Anchor Text:** Uses descriptive keywords (e.g., "Daily Bangla Newspapers" instead of "Click here").

## STEP 8: BACKLINK STRATEGY

1. **Directory Submissions:** Submit to local BD directories like `bd-directory.com`.
2. **Guest Posting:** Reach out to tech blogs in Bangladesh to write about "The Future of Digital Media in BD".
3. **Social Media:** Share every new listing on Facebook groups related to "Bangladeshi News".
4. **Broken Link Building:** Find dead links on Wikipedia's "List of newspapers in Bangladesh" and suggest BanglaNewsHub as a replacement.

---

## STEP 9: GOOGLE INTEGRATION

1. **Search Console:**
   - Go to [Google Search Console](https://search.google.com/search-console/).
   - Add your domain.
   - Verify ownership via DNS or HTML file.
   - Submit `/sitemap.xml`.
2. **Google Analytics:**
   - Create a GA4 property.
   - Add the tracking ID to your `index.html` or via a React hook.
